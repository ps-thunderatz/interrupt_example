# Interrupt Example

Esse repositório contém um exemplo de como utilizar pinos de input e de output.

## Descrição do cube e hardware

| Pino | Nucleo | Configuração           | Função |
|:---: |:---:   |:---                    |:---    |
| PB6  | D10    | GPIO EXTI6 (Pull-up)   | Ler bordas de subida e descida do sinal |
| PA5  | D13    | GPIO Output            | Acender um LED |
